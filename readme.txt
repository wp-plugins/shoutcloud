INTEGRATES YOUR BLOG WITH SHOUTCLOUD. EVERYTHING IS BETTER WHEN IT IS SHOUTED!

== Description ==

SHOUTCLOUD IS A MICROSERVICE THAT TRANSFORMS YOUR STRINGS TO UPPERCASE. THIS PLUGIN INTEGRATES ALL OF YOUR CONTENT
WITH THE SERVICE - ALL TITLES, CONTENT AND TAGS WILL BE CONVERTED.

ADVANTAGES:

* MICROSERVICES ARE THE FUTURE
* REDUCE SERVER LOAD BY DOING YOUR STRING MANIPULATION IN THE CLOUD
* BUILD ANTICIPATION BY DRASTICALLY INCREASING LOAD TIME

DISTADVANTAGES:

* NONE, THE CLOUD IS ALWAYS BETTER!

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory
2. Enjoy!

== Frequently Asked Questions ==

= What about images and links? Will this break them? =

You may be concerned that shouting URLs would break the case-sensitive links. Don't be! The plugin uses a smart filtering
system that removes all images and hyperlinks so that your audience can concentrate on the text.
